### Installation Steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/nattapoomS/tap-th.git
   cd tap-th
   ```

2. Download the video files from Google Drive links above

3. Place the videos in their respective directories:
   - `public/hero-video.mp4`
   - `app/vdo/0103(1).mp4`

4. Install dependencies:
   ```bash
   npm install
   ```

5. Run the development server:
   ```bash
   npm run dev
   ```

6. Open [http://localhost:3000](http://localhost:3000) in your browser
